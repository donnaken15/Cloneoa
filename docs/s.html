settings.ini

    [General]

  • Language(=0) – Play game in English (0)/Japanese (1)
  • HandyTitle(=0) – Display FPS counter and current level file.

    [Display]

  • FreeSize(=0) – Run the game at any resolution, does not work with Scale.
  • FullScreen(=0) – Run the game in fullscreen.
  • Scale(=2) – Change window scale. (Integer only)

    [Debug]

  • DebugDraw(=0) – Display information aiding in fixing bugs, etc.
  • RealtimeSrc(=0) – Run the game in fullscreen.
